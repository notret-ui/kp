# Дизайн-система «Печь.ру КП»

Единый источник правды по визуалу. Извлечён из Figma-шаблона «КП-200825-ДАО»
(fileKey `mhPk6so2Et1iviRisBmSHG`). Цвета на текущем этапе семплированы из PDF-рендера
и помечены «(уточнить по Figma-переменным)» — будут сверены через `get_variable_defs`,
когда снимется лимит Figma MCP.

Референсы: `reference/КП 200825 ДАО.pdf`, `reference/page1..4.png`, `reference/figma_overview.png`.

## Формат слайдов

- **Десктоп:** ландшафт 16:9, мастер 4000×2250 px (рендер веба — по ширине вьюпорта,
  PDF — landscape-страница 16:9).
- **Мобайл:** портрет, мастер 2250×4000 px. Тот же контент, перекомпонован вертикально.
- На каждом слайде — **красный сайдбар справа** с лого «Печь.ру», ссылкой
  «Подробнее на Печь.ру» и крупным полупрозрачным водяным знаком «печь.ру» + силуэт пламени.

## Типографика

- Гарнитура — уточнить по Figma (предположительно гротеск; зафиксируем при сверке переменных).
- Роли: H1 (заголовок слайда), H2 (название товара), body, caption (подписи к характеристикам),
  price-lg (крупная цена), price-strike (старая цена, зачёркнута).

## Компоненты-«кирпичи»

1. **Sidebar** — коралловая колонка справа: лого «Печь.ру» сверху, водяной знак,
   ссылка «Подробнее на Печь.ру» снизу. Присутствует на всех слайдах.
2. **ManagerBar** — плашка менеджера: круглый аватар, «Старший менеджер», имя,
   иконки соцсетей; рядом контактная пилюля (email + телефон).
3. **SpecTable** — «Характеристики:» в две колонки: подпись + значение
   (Высота/Глубина/Ширина/Вес/Мощность/… мм, кг).
4. **PriceBlock** — крупная коралловая цена + под ней зачёркнутая старая цена (если есть скидка).
5. **AdvantageItem** — иконка + заголовок + короткий текст (блок «Почему выбирают Печь.ру»).
6. **ContactCard** — серая карточка: иконка, заголовок, разделитель, значение(я).
7. **GalleryGrid** — сетка фото (2×3) «Фото наших довольных клиентов».
8. **ProductMiniCard** — мини-карточка сопутствующего товара: фото, название, цена, кнопка.

## Типы слайдов (из Figma)

1. **Обложка** — «Печь.ру — искусство тепла и уюта в каждом доме!» + 6 × AdvantageItem.
2. **О компании** — «Мы работаем с 2009 г.» + текст + фото.
3. **Карточка товара** — ManagerBar сверху; фото слева; название + SpecTable справа;
   PriceBlock снизу; «Описание товара» (длинный текст).
4. **Товары в этой категории** — ряд ProductMiniCard.
5. **Галерея** — GalleryGrid.
6. **Контакты** — 4 × ContactCard (многоканальный телефон, мессенджеры, email, время работы).

## Новый слайд (нет в Figma — проектируется в этом же стиле)

**«Итоги»** — таблица позиций (миниатюра, название, кол-во, цена, сумма),
отдельные строки доставки / монтажа / сопутствующих услуг, скидка на КП в целом,
строка **ИТОГО**. Тот же сайдбар, токены, типографика.
