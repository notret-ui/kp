# Дизайн-система «Печь.ру КП»

**Источник правды — проект на claude.ai/design «Печь.ру Design System»**
(projectId `cffe7775-571d-4414-8a6a-6fa40431bdac`), синхронизируется через DesignSync.
Там же — skill `pech-ru-design`, шрифты Museo (.otf), `assets/sprite.svg` (42 иконки),
UI-кит и preview-карточки компонентов.

Этот файл — выжимка под задачу КП. Значения ниже взяты из `colors_and_type.css`
дизайн-системы (часть значений в самом источнике помечена как реконструкция из UI-кита,
т.к. исходный `common.min.css` не прилагался — но это связная брендовая система).

КП-референсы Figma: `reference/КП 200825 ДАО.pdf`, `reference/page1..4.png`.

## Формат слайдов

- **Десктоп:** ландшафт 16:9, мастер 4000×2250 px.
- **Мобайл:** портрет, мастер 2250×4000 px.
- На каждом слайде — **оранжевый сайдбар справа** с лого «Печь.ру», ссылкой
  «Подробнее на Печь.ру» и полупрозрачным водяным знаком «печь.ру» + силуэт пламени.

## Типографика

- **Дисплейный:** `Museo Cyrl` (заголовки слайдов, названия, цена).
- **Текст/UI:** `Museo Sans Cyrl` (характеристики, подписи, кнопки, контакты).
- Шкала px: 11/12/13/14(base)/16/18/22/26/32/40/54. Line-height: tight 1.15 … loose 1.7.
- H1 40px/500, H2 32px/500, H3 26px/500; H5/H6 — uppercase, увеличенный трекинг.

## Прочие токены

- **Отступы:** 4/8/10/12/16/20/24/30/40/50/60/80 px.
- **Радиусы:** плоский стиль — 2/3/4 px (без pill-кнопок).
- **Тени:** редко (дропдауны, поповеры, ховер карточек), тёплый тон `rgba(44,28,22,…)`.
- **Ссылки:** пунктирное подчёркивание (`border-bottom: 1px dashed`).

## Правила бренда (из skill `pech-ru-design`)

- Только русский текст, формальное «вы», без эмодзи в клиентском UI.
- `&nbsp;` приклеивает короткие предлоги к слову (рус. типографика).
- Кнопки — строчными буквами. Тёплая «очаг/дерево/пламя» эстетика.

## Компоненты-«кирпичи» слайдов КП

1. **Sidebar** — оранжевая колонка справа: лого «Печь.ру», водяной знак, ссылка
   «Подробнее на&nbsp;Печь.ру». На всех слайдах.
2. **ManagerBar** — аватар, «Старший менеджер», имя, соцсети + пилюля (email + телефон).
3. **SpecTable** — «Характеристики:» в две колонки (Высота/Глубина/Ширина/Вес/Мощность…).
4. **PriceBlock** — крупная оранжевая цена + зачёркнутая старая (если есть скидка).
5. **AdvantageItem** — иконка (sprite) + заголовок + текст.
6. **ContactCard** — серая карточка: иконка, заголовок, разделитель, значение(я).
7. **GalleryGrid** — сетка фото 2×3.
8. **ProductMiniCard** — фото, название, цена, кнопка (сопутствующие товары).

Иконки берём из `assets/sprite.svg` (`#logo`, `cart`, `heart`, `phone`, чевроны, пламя, звёзды, соцсети).

## Типы слайдов

1. Обложка + 6 × AdvantageItem (+ имя клиента и дата).
2. О компании «с 2009 г.» + фото.
3. Карточка товара (ManagerBar, фото, SpecTable, PriceBlock, описание).
4. **Итоги** *(нет в Figma — проектируем в стиле системы):* таблица позиций
   (миниатюра, название, кол-во, цена, сумма), строки доставка/монтаж/услуги,
   скидка на КП, ИТОГО.
5. Галерея.
6. Товары в этой категории (ProductMiniCard).
7. Контакты (4 × ContactCard) + менеджер.
